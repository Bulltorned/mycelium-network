## What this is

A formal submission to the WIPO CWS Blockchain Task Force, to be presented at the WIPO Jakarta Regional Session in October 2026.

**Proposal:** a chain-neutral, token-free, WIPO-compatible specification for on-chain intellectual property evidence — **the Jakarta Protocol v1.0**. Reference implementation is Mycelium Network (five Solana programs, live on devnet, mainnet-beta pending audit).

**Anchor partner:** INFIA Group (120M+ audience, 36+ IPs including Dagelan, Tahilalats, Hai Dudu, Mindblowon Studio).

**Governance:** 3-of-5 Squads multisig, WIPO observer seat reserved, CC-BY 4.0 specification, MIT reference implementation.
