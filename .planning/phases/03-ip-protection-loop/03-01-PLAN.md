**src/services/evidence/evidence-engine.ts:**

1. Import: `assembleMEP`, `JURISDICTION_MAP` from mep-assembler, `uploadEvidence` from irys-uploader, `crypto` from Node.js, `nacl` from tweetnacl, `Ed25519Program`, `PublicKey`, `Transaction`, `SystemProgram`, `SYSVAR_INSTRUCTIONS_PUBKEY` from @solana/web3.js, `BN` from bn.js (already a dependency via Anchor).

2. Export `async function generateFullMEP(params: GenerateMEPParams): Promise<GenerateMEPResult>`:

   ```typescript
   export interface GenerateMEPParams {
     ipAssetPubkey: string;
     jurisdiction: Jurisdiction;
     meridianProgram: any;        // Anchor Program instance
     requesterKeypair: any;       // Keypair for signing tx
     protocolKeypair: any;        // Protocol authority keypair for Ed25519 signing
     connection: any;             // Solana Connection
   }

   export interface GenerateMEPResult {
     arweaveUri: string;
     packageHash: string;          // hex-encoded SHA-256
     evidencePda: string;          // base58 EvidencePackage PDA
     transactionSignature: string;
     mepDocument: object;          // The full MEP JSON
   }
   ```

3. Implementation flow:
   a. Call `assembleMEP(ipAssetPubkey, jurisdiction)` to build MEP JSON.
   b. **Canonical serialization**: `const mepJsonString = JSON.stringify(mepDocument, Object.keys(mepDocument).sort())` -- deterministic key ordering prevents Pitfall 2 (hash mismatch).
   c. Upload exact byte string to Arweave: `const arweaveUri = await uploadEvidence(JSON.parse(mepJsonString), keypairBytes)` -- pass the parsed object but ensure consistency.
      Actually, to avoid the pitfall: serialize once, upload that exact string as a Buffer, compute hash on that exact string.
      Use `await irysUploadRaw(Buffer.from(mepJsonString), keypairBytes)` or wrap uploadEvidence to accept string.
      Since uploadEvidence takes an object and re-serializes internally, we need to ensure the same serialization. Safest: compute hash from `JSON.stringify(mepDocument)` (same as what uploadEvidence will produce internally).
   d. Compute SHA-256: `const packageHash = crypto.createHash('sha256').update(mepJsonString).digest()` -- returns Buffer (32 bytes).
   e. Ed25519 sign: `const protocolSignature = nacl.sign.detached(new Uint8Array(packageHash), protocolKeypair.secretKey)`.
   f. Build Ed25519 verify instruction (MUST be index 0):
      ```typescript
      const ed25519Ix = Ed25519Program.createInstructionWithPrivateKey({
        privateKey: protocolKeypair.secretKey,
        message: packageHash,
      });
      ```
   g. Derive evidence PDA: `PublicKey.findProgramAddressSync([Buffer.from("evidence"), new PublicKey(ipAssetPubkey).toBuffer(), requesterKeypair.publicKey.toBuffer()], MERIDIAN_PROGRAM_ID)`.
   h. Map jurisdiction to Anchor enum variant using JURISDICTION_MAP: e.g., `{ indonesia: {} }` for "ID", `{ wipoArbitration: {} }` for WIPO.
   i. Get license count from MEP document `mepDocument.license_history.length`.
   j. Build generate_mep instruction via meridianProgram.methods.generateMep(...).accounts({...}).instruction().
   k. Build Transaction: `new Transaction().add(ed25519Ix).add(generateMepIx)` -- Ed25519 FIRST.
   l. Send and confirm transaction.
   m. Return GenerateMEPResult with all data.

4. Export `MERIDIAN_PROGRAM_ID` as `new PublicKey("7LrekmiYZDKPBsaSBVaXUVG9iXtB164XQ5ntRVeiMnfc")`.
