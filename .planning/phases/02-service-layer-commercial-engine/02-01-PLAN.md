<objective>
Build all off-chain infrastructure services that the commercial engine (Plan 02-02) and MCP completion (Plan 02-03) depend on: PostgreSQL database with connection pool, Helius webhook-driven indexer, Irys/Arweave upload pipeline, and BIP-44 per-agent key vault.

Purpose: Infrastructure unblocks everything downstream. Without PostgreSQL indexing, license verification cannot query existing licenses at scale. Without Irys, metadata cannot be permanently stored. Without key vault, agents share a single keypair (security disaster).

Output: 9 new service modules in src/services/, PostgreSQL schema DDL, updated package.json with new dependencies.
</objective>
